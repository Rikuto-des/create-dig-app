# レイアウトルール

## 基本レイアウトパターン

### 管理画面（Admin）
- SideBar（固定幅 240px）+ メインコンテンツ（残り幅）
- メインコンテンツは `PageLayout` で囲む
- SideBar は常に表示

### ユーザー向け
- ヘッダー（固定高さ）+ フルワイドコンテンツ
- ナビゲーションはヘッダー内に配置

### LP
- フルワイド + セクション構成
- セクションごとに背景色を交互に変える

## スペーシング

| 用途 | Tailwind クラス |
|---|---|
| セクション間 | `py-8` or `py-12` |
| カード間 | `gap-4` or `gap-6` |
| カード内パディング | `p-4` or `p-6` |
| テキスト間 | `space-y-2` |
| ページ全体パディング | `px-8 py-6` |

## レスポンシブ

| ブレークポイント | Tailwind prefix | 用途 |
|---|---|---|
| 640px | `sm:` | モバイル → タブレット |
| 768px | `md:` | タブレット → デスクトップ |
| 1024px | `lg:` | デスクトップ（メイン） |
| 1280px | `xl:` | ワイドスクリーン |
