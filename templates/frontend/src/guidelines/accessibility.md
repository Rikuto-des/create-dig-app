# アクセシビリティガイドライン

## 基本ルール

- すべてのインタラクティブ要素にキーボードフォーカスが当たること
- 色だけで情報を伝えない（アイコンやテキストも併用）
- コントラスト比 4.5:1 以上を確保（WCAG AA レベル）
- 画像には `alt` テキストを付与
- フォームの入力欄には `label` を紐付ける

## ボタン
- `disabled` 状態は `opacity-50 cursor-not-allowed` で視覚的に区別
- アイコンのみのボタンには `aria-label` を付与

## テーブル
- `<th>` に `scope` 属性を指定
- 空の状態は「データがありません」等のメッセージを表示

## カラー
- `primary-500` をテキスト背景に使う場合は白文字を使用（コントラスト比確認済み）
- `error` / `success` / `warning` カラーは色覚多様性を考慮し、テキストやアイコンと併用

## フォーカス
- `focus:ring-2 focus:ring-primary-500 focus:ring-offset-2` を統一のフォーカススタイルとして使用
